# Android_TeachingSW
2020.08 ~ 2020.12 까지 개발한 ETRI(한국전자통신연구원) 협동로봇용 티칭 소프트웨어입니다. 소스코드는 상업용이므로 공개할 수 없는점 양해 부탁드립니다.

> 개발 기간: 2020.08 ~ 2020.12

> 개발 환경 :  
Android Studio Version : 4.0.1  
Compile Sdk Version : 28  
Min Sdk Version : 23
> 개발 언어 : Kotlin
